Guide content

The following guide explains how to use WebApp to set questions for LocAsk mobile app. The guide contains:

- System requirements;

- How to start the app;

- How the app works.

The app is only developed for technical users who are responsible for managing the data accessed by LocAsk. It should not be made public or accessed by final users who may enter unappropriate or wrong information into the app.

System requirements

The app necessitates a web-browser to be run. It works on Google Chrome, Microsoft Internet Explorer, Microsoft Edge, Safari, Opera and Mozilla Firefox.

In order to load the webpage, the user must either run the app from a university computer centrally connected to the UCL LAN or alternatively be connected to the UCL VPN. UCL members can download Cisco AnyConnect to establish the VPN connection; further information on how to connect to the UCL VPN can be found at the following link.

How to start the app

The app does not require any installation and is available at the WebApp URL web page

Be sure to be connected to the UCL VPN and start the NodeJSserver and the Phonegap server using a SSH software such as Bitvise, Cyberduck or Fugu. The connection is established using credentials:

Host: developer.cege.ucl.ac.uk

SSH Port: 30095

Username: studentuser

Password: XXXXXX - contact the administrator for the password

Once the connection is established, the user must type the following commands in the terminal window to start the server:

- cd /home/studentuser/code (press enter)

- node NodeJSserver.js (press enter - the server is now running and data can be uploaded/downloaded)

- cd /home/studentuser/code/WebApp/ucesegr (press enter)

- sudo su (press enter and enter the admin password)

- phonegap serve (press enter - the Phonegap server is now running)

The user can now access WebApp web page to enter new questions.

How the app works

The app has been designed to be used on a computer and although it can also be accessed from mobile devices, it could be less convenient to operate.

The page displays a map which can be explored to find new possible question locations. The coordinates of a specific point can be accessed by clicking on the desired location and copying the lat long values displayed in the pop-up window. The coordinates can then be pasted in the respective fields in the question form.

The left-side menu shows two options:

Add a question: the button redirects the user to the question form to add a new question;

Load Points: the button loads the points of interest already uploaded by the server on the database.

The question form contains all the necessary fields to set a question. Once information has been inserted, clicking on the Upload data button will will record the question on the database. To set up another question, either refresh the page or go back to the map, check the newly uploaded point" and open the question form again.
